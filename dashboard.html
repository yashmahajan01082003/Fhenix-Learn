Integration

Copy this code into your application to authenticate and retrieve admin dashboard data.

const loginResponse = await fetch('http://localhost:3101/api/admin/login', { method: 'POST', headers: { 'Content-Type': 'application/json' }, body: JSON.stringify({ username: 'admin', password: 'fhenix2024' }) }); const { token } = await loginResponse.json(); localStorage.setItem('adminToken', token);
const dashResponse = await fetch('http://localhost:3101/api/admin/dashboard', { headers: { 'X-Admin-Token': token } }); const data = await dashResponse.json(); console.log(data);
Recommendation: store tokens securely and use environment variables for credentials.
