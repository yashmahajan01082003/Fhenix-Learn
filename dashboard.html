Request Body:
{ "username": "admin", "password": "fhenix2024" }
Response (Success):
{ "success": true, "token": "abc123def456ghi789jkl012mno345pqr678" }
Response (Error):

📊 GET /api/admin/dashboard

GET /api/admin/dashboard

Fetch all analytics data including aggregates, charts data, and complete user information. Requires authentication token.

Headers Required:
X-Admin-Token: {your_session_token}
Response Structure:
{ "aggregates": { "totalUsers": 150, "totalXP": 500000, "avgXP": 3333, "totalBadges": 450, "totalMintedBadges": 320, "totalBadgeHolders": 120, "mintedPercentage": 71, "topModules": [...], "topLessons": [...], "xpDistribution": {...} }, "users": [ { "user_id": "user_123", "display_name": "John Doe", "xp": 5000, "badges_count": 3, "modules_completed": 12, "lessons_completed": 45, "wallet_address": "0x1234...", "last_tx_hash": "0xabcd..." }, ... ] }
cURL Example:
curl -X GET http://localhost:3101/api/admin/dashboard \ -H "X-Admin-Token: your_token_here"
ℹ️ How to Get a Token:
1. First, use the POST /api/admin/login endpoint (above) to get a token 2. Copy the token from the response 3. Paste it in the Admin Token field below 4. Click "Fetch Dashboard Data"

📚 Integration Guide

Step 1: Login

const loginResponse = await fetch('http://localhost:3101/api/admin/login', { method: 'POST', headers: { 'Content-Type': 'application/json' }, body: JSON.stringify({ username: 'admin', password: 'fhenix2024' }) }); const { token } = await loginResponse.json(); localStorage.setItem('adminToken', token);

Step 2: Fetch Dashboard Data

const dashResponse = await fetch('http://localhost:3101/api/admin/dashboard', { headers: { 'X-Admin-Token': token } }); const data = await dashResponse.json(); console.log(data.aggregates); console.log(data.users);

Environment Variables:

ADMIN_USERNAME=admin ADMIN_PASSWORD=fhenix2024
